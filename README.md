# Memory Engine Graph Template App

## Project Summary
The **Memory Engine Graph Template App** is a sophisticated template for building graph-based memory systems. It leverages modern web technologies to provide a highly interactive and scalable interface for visualizing and managing complex data relationships.

## Tech Stack
- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Graph Visualization**: [React Flow](https://reactflow.dev/)
- **Embeddings**: [@xenova/transformers](https://huggingface.co/docs/transformers.js/index) (Local ONNX embeddings)
- **Database**: [PostgreSQL](https://www.postgresql.org/) / [Supabase](https://supabase.com/) with [pgvector](https://github.com/pgvector/pgvector)
